# FluxerThemeViewer

## About the Project

- It's not an app, just a website that let's you view some themes I made
- All themes that include animations or 'advanced' css are in the 'Advanced.' so just click "Advanced" from the list of filters to see those

## How to Use 

- go to https://picksol.github.io/FluxerThemeViewer/
- copy the CSS code from the button
- paste it into your settings in Fluxer
